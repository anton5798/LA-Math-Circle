\begin{sol} 2000. Note that $2019 = 3 * 673$, and both $3$ and $673$ are primes. Moreover,
$$n^2 + 20n + 19 = (n+1)(n+19)$$
As $2019$ is divisible by $3$, one of the factors $(n+1)$ and $(n+19)$ must be divisible by $3$. Since the difference between them is divisible by $3$, both of them must divisible by $3$. Moreover, one of them also has to be divisible by $673$, which means that this number has to be divisible by $3*673 = 2019$. The smallest $n$ for which this condition holds is $2000$.
\end{sol}
